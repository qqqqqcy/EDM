## API

| 属性      | 说明                               | 类型                                    | 默认值        | 必填    |
| --------- | ---------------------------------- | --------------------------------------- | ------------- | ------- |
| theme     | 按钮主题。自定义按钮颜色时请勿传值 | `'primary' | 'secondary' | 'customize'` | `'customize'` | `false` |
| size      | 尺寸                               | `'s' | 'm' | 'l'`                       | `'l'`         | `false` |
| ghost     | 幽灵按钮                           | `boolean`                               | `false`       | `false` |
| inline    | 行内按钮                           | `boolean`                               | `false`       | `false` |
| rectangle | 矩形按钮                           | `boolean`                               | `false`       | `false` |
