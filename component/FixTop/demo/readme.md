## API

| 属性          | 说明                                 | 类型      | 默认值 | 必填    |
| ------------- | ------------------------------------ | --------- | ------ | ------- |
| visible       | description                          | `boolean` | -      | `false` |
| showLeftIcon  | 是否显示关闭按钮                     | `boolean` | -      | `false` |
| leftIcon      | 可以在左侧展示特定图标               | `any`     | -      | `false` |
| showCloseIcon | 是否显示关闭按钮                     | `boolean` | -      | `false` |
| closeIcon     | 关闭按钮的图标，不传展示一个标准的 X | `any`     | -      | `false` |
